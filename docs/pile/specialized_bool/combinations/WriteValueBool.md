# `WriteValueBool` — adds `setTrue()`/`setFalse()` to `WriteValue<Boolean>`

Source folder: `src`. Package: `pile.specialized_bool.combinations`.

Narrows [`WriteValue`](../../aspect/WriteValue.md) to `Boolean` and adds two boolean write conveniences as default methods: `setTrue()` calls `set(Boolean.TRUE)` and `setFalse()` calls `set(Boolean.FALSE)`. No other new members.
