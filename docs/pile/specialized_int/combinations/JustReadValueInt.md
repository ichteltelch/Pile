# `JustReadValueInt`

Pure stub interface narrowing [`JustReadValue`](../../aspect/JustReadValue.md) to `Integer`; adds no new members beyond what `ReadValueInt` and `JustReadValueComparable<Integer>` supply.

Source folder: `src` · package `pile.specialized_int.combinations`.

Extends `ReadValueInt` and `JustReadValueComparable<Integer>`. The `@FunctionalInterface` annotation lives on the generic `JustReadValue`; this interface is a typed assembly that makes integer-producing lambdas assignment-compatible with the reactive graph.
